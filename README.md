The final dashboard provides a clear overview of the data profession landscape. It highlights that Data Scientists tend to have the highest average salaries, while roles like Data Analyst and Data Engineer show steady growth across countries. The United States and the United Kingdom emerge as leading regions for data-related opportunities. Across all roles, Python, SQL, and Power BI stand out as the most in-demand technical skills, reaffirming their importance in the data industry.

This project demonstrates how Power BI can be used for career and industry analysis by turning raw data into interactive and visually rich insights. It also strengthened my understanding of data modeling, relationship management, and visualization best practices.
